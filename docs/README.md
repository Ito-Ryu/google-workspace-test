# Docs 概要

この `docs` ディレクトリは、Google Workspace における Google Group の分散管理および Terraform による自動化に関するドキュメントを格納しています。大きく分けて、システムの設計と思想（Why / What）を説明する `design_doc` と、実際の構築・運用手順（How）を説明する `guide` の2つに分かれています。

## ディレクトリ構成

| ディレクトリ | 内容 | 役割 (Role) |
| :--- | :--- | :--- |
| `design_doc/` | システム設計、アーキテクチャ、命名規則、制約 | 「なぜそのように設計されたのか (Why / What)」の定義 |
| `guide/` | 管理者(SRE)および利用者の構築・運用手順 | 「実際にどう作業を行うのか (How)」の定義 |
