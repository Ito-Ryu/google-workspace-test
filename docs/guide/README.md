# Guide: セットアップおよび運用手順書 (Setup & Operations Guides)

このディレクトリ（`docs/guide`）には、分散型 Google Group 管理システムを利用するための、管理者（SRE）および利用者（プロジェクトチーム）向けの手順や運用ルールが格納されています。設計思想やアーキテクチャに基づいて、具体的に**「どのように作業を行うか（How）」** を記述しています。

## 格納されているドキュメント

| ファイル名 | 概要 | 対象者 |
| :--- | :--- | :--- |
| `setup.md` | 管理者向けセットアップ手順 | 管理者 / SRE |
| `sop.md` | プロジェクト向け標準作業手順書 | 各プロジェクトチーム / 利用者 |
